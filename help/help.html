Help - tobCV

More and more recruiters combing the net to find talented people. This, tobCV, is a html template that will help you set up your own personal online CV. Hopefully some recruiter will come by and spot you!

Included - root folder/

  1. index.html - The one page included in this template
    • The one page included in this template
  2. assets/css/styles.css
    • The styles for index.html
  3. assets/css/print-styles.css
    • The styles for the printable version of index.html
    • Only containing styles that differs from screen version of index.html
  4. assets/img/
    • headerEdge.png - Used to produce the "over the edge effect" for the section headers
    • portrait.png - Just an example image of a portrait. Replace this with your own portrait
    • shadow.png - Used to produce the shadow effect separating the header from the body
    • skillOverlay.png - Used to produce the pointer for the skill bar
    • darkStripes.png - Used as background image for the headers of work- and education sections
