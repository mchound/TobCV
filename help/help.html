Help - tobCV

More and more recruiters combing the net to find talented people. This, tobCV, is a html template that will help you set up your own personal online CV. Hopefully some recruiter will come by and spot you!

Included - root folder/

  1. index.html
    • The one page included in this template
  2. assets/css/styles.css
    • The styles for index.html
  3. assets/css/print-styles.css
    • The styles for the printable version of index.html
    • Only containing styles that differs from screen version of index.html
  4. assets/img/
    Image Description Example
    headerEdge.png Used to produce the "over the edge effect" for the section headers. Some, partly transparent, sections makes the background color shine through
    portrait.png Just an example image of a portrait. Replace this with your own portrait
    shadow.png Used to produce the shadow effect separating the header section from the body section
    skillOverlay.png Used to produce the pointer for the skill bar. Some, partly transparent, sections makes the background color shine through
    darkStripes.png Used as background image for the headers of work- and education section
    stripes.png Used as background image for the top bar. The stripes are partly transparent to let the background color shine through
  5. assets/js/main.js
    • Initialization of scripts running when the dom has finished loading. More info in the scripts section
  6. assets/external/jquery/js/jquery-1.8.2.min.js
    • minified version of jquery 1.8.2
  7. assets/external/tobPlugins/tobPopoverJS, See jquery plugin section for more info
    • /css/styles.css - Styles for elements in jquery plugin tobPopoverJS.
    • /js/tobPopoverJS.jss - Scripts for jquery plugin tobPopoverJS.
  8. assets/external/tobTimelineJS/tobTimelineJS, See jquery plugin section for more info
    • /css/styles.css - Styles for elements in jquery plugin tobTimelineJS
    • /js/tobTimelineJS.jss - Scripts for jquery plugin tobTimelineJS.
    • /img/timelineBg.png - Background image used in jquery plugin tobTimelineJS.
